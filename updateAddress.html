Updating the network address

This will allow you to access the database if automatic updates changed the directory names.

Sometimes, automatic updates change the directory names slightly. If you are having trouble logging in, you can tell the computer to search for the 4D database.

  1. Open the 4D client.
  2. Select the PCT/IP tab. This is the second tab.
  3. Select Soils: 4DC and hit OK.
  4. Log in as usual. See Log into the 4D database
