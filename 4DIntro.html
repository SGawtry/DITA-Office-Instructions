Introduction to 4D

4D, short for 4th Dimension, is the database software used at the Soil Testing and Research Analytical Lab. It has been in use since 1998, with the latest software update occuring in 2003. While it still produces accurate reports, there are some quirks that users must manage when working with the database. These files will overview common functions you will complete in 4D. There will also be some troubleshooting guidelines.
